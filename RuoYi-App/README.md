### 小程序说明

#### 基于uniapp, 引入了 uView 作为开发框架

- [uView官网](https://xuqu.gitee.io/components/intro.html)

#### 初始化: 在HBuilderX中打开终端运行

```bash
npm install
```

#### 框架已配置, 直接使用即可

- 例如

```html
<template>
	<u-button text="按钮"></u-button>
</template>
```
